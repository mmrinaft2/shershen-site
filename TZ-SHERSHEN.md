## 4. CRO (КОНВЕРСІЙНА ОПТИМІЗАЦІЯ)

### 4.1 Поточний аналіз конверсійної воронки

```
Візит → Hero CTA → Скрол → Послуги → FAQ → Контактна форма
         ↓                                      ↓
    "Консультація"                         Заповнення форми
         ↓                                      ↓
    Scroll до форми                        "Відправлено" (fake)
```

**Проблеми:**
1. Форма не відправляє реальних даних (тільки визуальний фідбек)
2. Немає альтернативних CTA (Telegram бот, Calendly)
3. Немає exit-intent попапу
4. Немає соціального доказу (відгуки, кейси, логотипи)
5. Телефон — placeholder, знижує довіру

### 4.2 Обов'язкові CRO-зміни

#### A. Trust bar (логотипи клієнтів)
**Розташування**: між Hero та Services
```
Нам довіряють: [Logo1] [Logo2] [Logo3] [Logo4] [Logo5] [Logo6]
```

#### B. Testimonials (відгуки)
**Розташування**: після Services
```
"Шершень створив нам AI-бота за 2 тижні. Кількість звернень у підтримку
знизилась на 40%."
— Олена К., CEO, CompanyName
```
Мінімум 3 відгуки з реальними фото та компаніями.

#### C. Sticky CTA на мобільному
Фіксована кнопка "Консультація" внизу екрану при скролі.

#### D. Exit-intent popup
```
Зачекайте! 🐝
Отримайте безкоштовний PDF:
"5 способів, як AI може зекономити вашому бізнесу $10 000/міс"
[Email] [Завантажити]
```

#### E. Форма — бекенд
Підключити реальну відправку:
- Варіант 1: Formspree / Getform (безкоштовно до 50 заявок/міс)
- Варіант 2: Google Sheets API
- Варіант 3: Telegram Bot API (заявки одразу в Telegram)

#### F. Альтернативні CTA
- "Написати в Telegram" (прямий лінк на бота)
- "Забронювати дзвінок" (Calendly embed)
- WhatsApp кнопка (floating)

### 4.3 A/B тести (після запуску)

| Тест | Варіант A (контроль) | Варіант B | Метрика |
|---|---|---|---|
| Hero CTA | "Безкоштовна консультація" | "Отримати AI-стратегію за 0 грн" | CTR |
| Hero subtitle | Поточний текст | "Вже 50+ компаній автоматизували бізнес з нами" | Scroll depth |
| Pricing | Сторінка цін | Калькулятор вартості | Lead quality |
| Form fields | 4 поля | 2 поля (ім'я + телефон) | Completion rate |
| Trust bar | Без | З логотипами | Form submissions |

## 6. GEO (ВИДИМІСТЬ В AI-ПОШУКУ)

### 6.1 Answer-first структура

Кожна сторінка послуги повинна починатися з прямої відповіді:
```
❌ Зараз: "Розробляємо інтелектуальних чат-ботів..."
✅ Потрібно: "AI-асистент для бізнесу — це інтелектуальний чат-бот,
який автоматизує комунікацію з клієнтами через Telegram, Viber, WhatsApp
та ваш сайт. Шершень розробляє такі рішення від 2 тижнів, з інтеграцією
в CRM та підтримкою 50+ мов."
```

### 6.2 H2 як питання

```
❌ Зараз: "Що ми робимо"
✅ Потрібно: "Які AI-рішення ми створюємо для бізнесу?"

❌ Зараз: "Як ми працюємо"
✅ Потрібно: "Як відбувається впровадження AI у ваш бізнес?"

❌ Зараз: "Чому Шершень"
✅ Потрібно: "Чому компанії обирають Шершень для AI-рішень?"
```

### 6.3 Entity identification

На кожній сторінці чітко ідентифікувати:
- **Хто**: "Шершень — українська AI-студія, заснована у 2023 році"
- **Що**: "Розробка AI-асистентів, впровадження ШІ, SEO-контент"
- **Де**: "Працюємо з компаніями по всій Україні та Європі"
- **Технології**: "GPT-4, Claude, LangChain, RAG, Runway Characters"
